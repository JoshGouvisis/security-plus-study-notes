# 4.8 Security Monitoring Tools

* **Packet Capture:** Using tools like Wireshark to inspect the actual contents of network traffic for deep-dive analysis and troubleshooting.
* **NetFlow:** Analyzing traffic patterns, volume, and "who is talking to whom" (IP conversations) rather than looking at the packet payload.
* **Dashboards:** Real-time SIEM visualizations (graphs, charts, and maps) that provide an "at-a-glance" status of the security environment.
* **Log Aggregation:** Utilizing a SIEM to collect and correlate data from diverse sources to identify complex attack patterns.
