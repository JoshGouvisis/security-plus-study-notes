# 5.1 Governance and Risk
* **Security Governance:** The system by which an organization’s security program is directed and controlled.
* **Roles & Responsibilities:** * **CISO:** Strategic leadership and accountability.
    * **Security Manager:** Tactical implementation.
    * **Custodian:** Technical maintenance of data.
* **Frameworks:** NIST Cybersecurity Framework (CSF) and ISO 27001.
