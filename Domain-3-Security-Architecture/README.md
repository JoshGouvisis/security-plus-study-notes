# Domain 3 — Security Architecture (18%)

![Status](https://img.shields.io/badge/Status-In_Progress-orange?style=flat)

## Topics Covered
* Security Control Categories (Technical, Managerial, Operational, Physical)
* Network Design (Segmentation, Zero Trust, Port Security)
* Data Protection (Encryption, Masking, Hashing, DLP)
* Resilience (High Availability, Backups, Recovery Sites)

## 🎯 Domain Overview
This domain focuses on the design and implementation of secure enterprise architectures. It covers the selection of security controls, the application of secure network principles, the strategies used to protect data at rest and in transit, and the planning required to ensure organizational resilience.

## 📝 Objective Progress
- [ ] 3.1 Compare and contrast types of security controls.
- [ ] 3.2 Given a scenario, apply secure network design concepts.
- [ ] 3.3 Compare and contrast concepts and strategies to protect data.
- [ ] 3.4 Explain the importance of resilience and recovery in security architecture.
