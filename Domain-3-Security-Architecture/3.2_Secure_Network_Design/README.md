
# 3.2 Secure Network Design

Architecture models define how traffic flows and how trust is managed.

## Infrastructure Concepts
* **Infrastructure as Code (IaC):** Defining servers and networks via code to ensure a "perfect version" every time.
* **Serverless (FaaS):** Applications separated into individual functions that run in ephemeral containers.
* **Microservices & APIs:** Breaking monolithic applications into smaller, interconnected services.

## Network Architecture
* **Segmentation:** Dividing the network into smaller parts to prevent lateral movement.
* **Zero Trust:** "Never trust, always verify." Requires continuous authentication regardless of location.
* **Port Security:** Disabling unused ports or using 802.1X to prevent unauthorized physical connections.
